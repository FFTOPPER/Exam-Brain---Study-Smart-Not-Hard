# Exam Brain 🧠

**Study Smart, Not Hard**

## 🚀 Problem

Students waste time studying everything without knowing what actually matters for exams.

## 💡 Solution

Exam Brain analyzes past question papers and tells students:

* What to study first
* What to skip
* What is likely to appear in exams

## ⚙️ Features

* Upload past papers (PDF/DOCX)
* Extract real topics from questions
* Identify most important topics
* Generate practice questions
* Smart study plan
* Last Night Mode (quick revision)

## 🧠 Tech Stack

* Frontend: HTML/CSS/JS
* Backend: Python (FastAPI)
* AI: OpenAI API
* Charts: Chart.js

## 🎯 How it works

1. Upload question papers
2. System extracts topics (T1, T2…)
3. Counts frequency
4. Ranks topics
5. Generates questions + plan
