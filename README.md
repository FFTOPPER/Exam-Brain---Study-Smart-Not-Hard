## Problem Statement

Students often prepare for exams using scattered materials such as PDFs, notes, and previous question papers without a clear strategy.

Although past papers contain valuable insights, students face difficulties in:

* Identifying frequently asked topics
* Understanding question patterns
* Prioritizing high-impact areas

This leads to inefficient preparation, wasted time, and missed scoring opportunities.

---

## Solution

Exam Brain is an AI-powered system that analyzes past question papers and converts them into clear, actionable study guidance.

The system:

* Extracts real topics from documents
* Identifies how often each topic appears
* Ranks topics based on importance
* Generates practice questions
* Creates a structured study plan

It acts as a decision-support tool for exam preparation.

## How It Works

1. Upload past question papers
2. Extract text from documents
3. Identify topics and map them to questions
4. Count how frequently each topic appears
5. Rank topics based on importance
6. Generate practice questions and study plans
7. Display results in a structured interface
